== Description == 

WS Paypal Donation plugin provides to add paypal donation button to your blog.

<strong>Features</strong>

* Easy to install
* Simple to use
* Customize Donate Button

<strong>Shortcode</strong>

`[ws_paypal_donation]`

`<?php echo do_shortcode( '[ws_paypal_donation]' ); ?>`

== Installation ==

1. Upload WS Paypal Donation to the  /wp-content/plugins/ directory
2. Activate the WS Paypal Donation plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings > WS Paypal Donation
4. Set Options and Save
